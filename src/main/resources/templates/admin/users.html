User Management Guidelines:
Student Accounts
  • Must have a valid student ID
  • Can view own attendance and class information
  • Cannot book rooms directly
  • Access limited to own data
Teacher Accounts
  • Must have a valid staff ID
  • Can mark attendance for their classes
  • Can book rooms for their classes
  • Manage their class schedules
Administrator Accounts
  • Full system access and management
  • Can manage all users and classes
  • View all attendance and booking data
  • System configuration and maintenance
